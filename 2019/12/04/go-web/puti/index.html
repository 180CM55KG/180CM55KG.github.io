源码安装

Go语言下载与环境变量配置

关于Go语言在linux Ubantu中的下载配置有很多教程，但当我使用那些教程的时候还是遇到了很多问题，因此在这里我自己在写一遍教程

首先是Go语言的下载，Go

选择适合你系统的Go语言下载，本次项目要求Go语言的版本在1.11以上（以下的还真用不了），1.11以上有Go module

下载过后，我们把它解压到 /usr/local/ 目录下

tar -C /usr/local -xzf go1.13.4.linux-amd64.tar.gz

接下来就是配置环境变量, 在/etc/profile （即全局环境变量下修改）最后添加

vim /etc/profile

export GOROOT=/usr/local/go
export GOPATH=$HOME/go #这里是你go的workspace
export PATH=$PATH:$GOROOT/bin


source /etc/profile

立刻生效

这里可能会出现一个问题，就是关闭终端后，环境变量就失效了，在网上查了半天后，我的解决方法是这样的

vim ~/.bashrc

在里面添加

export GOPATH=$HOME/go
export GOROOT=/usr/local/go
export GOARCH=386
export GOOS=linux
export GOTOOLS=$GOROOT/pkg/tool
export PATH=$PATH:$GOROOT/bin:$GOPATH/bin


go version //检测安装是否成功

到此安装Go就算完成了

Git下载

最简单的下载方式

apt-get install git

这种安装方式不是很好

其他安装方法也百度得到，这里就不写了

源码下载

# 下载
$ go get -u github.com/puti-projects/puti

# 使用Makefile来构建程序
$ cd $GOPATH/src/github.com/puti-projects/puti
$ make

本次安装是在我的Ubantu16.04的虚拟机上，windows下出了很多问题，浪费了我很多时间。

MySQL下载

这个项目需要mysql数据库来运行

MySQL的安装教程 https://blog.csdn.net/james_nan/article/details/82053430

接下来就是把puti下面的sql文件导入到数据库中

先建好数据库

$ mysql -uroot -p
$ create database db_puti

导入sql文件

$ cd $GOPATH/src/github.com/puti-projects/puti/init/scheam/

$ mysql -uroot -p -B db_puti < db_puti.sql

修改配置文件

配置文件位于config目录

我们新建一个config.yaml文件，在里面写入

runmode: release                 # 开发模式, debug, release, test
addr: 8000                  # HTTP绑定端口
name: puti              # API Server的名字
ping_url: http://127.0.0.1:8000   # pingServer函数请求的API服务器的ip:port
ping_max_num: 10           # pingServer函数try的次数
jwt_secret: putidemojwtseCrect013keyideO
gormlog: false

# tls
tls:
  https_open: false
  addr: 8080
  cert: configs/server.crt
  key: configs/server.key

# logger
log:
  logger_file: logs/server.log
  logger_max_size: 1024 
  logger_max_backups: 7
  logger_max_age: 31 

# database
db:
  name: db_puti
  addr: 127.0.0.1:3306
  username: root
  password: root

启动项目

$ go run main.go
